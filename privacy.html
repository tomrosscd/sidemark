Sidecar Privacy Policy

Effective date: 2 July 2026  |  Last updated: 2 July 2026

Sidecar is a Chrome extension that helps people work more effectively with Shopify Sidekick, Shopify's built in AI assistant. This policy explains what the extension can access, what it does with that information, and what it deliberately does not do.

Sidecar is published by Thomas Ross, a sole trader based in Melbourne, Australia ("we", "us", "our"). If you have any questions about this policy, contact us at tom@convertdigital.com.au.

The short version

Sidecar runs entirely in your browser. It does not collect, store, sell, or transmit your Sidekick conversations or any personal information to us or to any third party. There is no account, no analytics, and no tracking. The only network request the extension makes is to download its own prompt library.

What Sidecar can access

Sidecar operates only within the Shopify admin (admin.shopify.com). When you actively use it, it can access two things:

  • The content of your active Shopify admin tab. When you click Insert, Copy, or Export, Sidecar reads the Sidekick conversation on that tab so it can insert a prompt into the input field or convert the conversation to Markdown.
  • Your local extension storage. Sidecar keeps a cached copy of its prompt library so the side panel opens instantly.

Sidecar only acts in response to an explicit action you take. It does not read or monitor pages in the background, and it does nothing on websites outside the Shopify admin.

How that information is used

Everything Sidecar does with the conversation content happens locally on your device and only at the moment you request it:

  • In Prompts mode, a prompt you select is inserted into the Sidekick input field.
  • In Export mode, the current Sidekick conversation is converted to clean Markdown, which is placed on your clipboard or saved as a local file for you to keep, share, or paste elsewhere.

The conversation content is never sent to us or to any external server. Once the action is complete, Sidecar does not retain a copy of it.

What Sidecar does not do

  • It does not collect or transmit your Sidekick conversations, prompts, or personal data to us or anyone else.
  • It does not use analytics, tracking pixels, cookies, fingerprinting, or advertising.
  • It does not sell or share any data. There is no data to sell or share.
  • It does not require an account or ask for login credentials.
  • It does not run on websites other than the Shopify admin.

Local storage

Sidecar uses your browser's local extension storage (chrome.storage.local) to cache its prompt library and a small amount of interface state, such as your last selected category. This data stays on your device, is not readable by us, and is removed when you uninstall the extension or clear it through your browser. It never contains your conversation content.

Network activity

Sidecar makes a single type of network request: it downloads its prompt library as a static JSON file from https://raw.githubusercontent.com/tomrosscd/sidecar/main/prompts.json, hosted on GitHub. This request contains no personal information and none of your conversation content. It is a standard file download, in the same way your browser downloads any web resource.

As with any request over the internet, the host serving that file (GitHub) may log standard technical information such as your IP address and request time in its own server logs, under GitHub's privacy statement. We do not have access to those logs and do not receive that information.

Permissions we request, and why

Sidecar requests the minimum permissions needed to function:

  • activeTab and scripting: to insert prompts into, and read the conversation from, the Shopify admin tab you are actively working in, only when you trigger an action.
  • Host access to admin.shopify.com: to operate within the Shopify admin where Sidekick lives.
  • sidePanel: to display Sidecar's interface in the browser side panel.
  • storage: to cache the prompt library locally so the panel loads quickly.

Compliance with the Chrome Web Store Limited Use policy

Our use of information accessed through Sidecar complies with the Chrome Web Store User Data Policy, including its Limited Use requirements. Sidecar accesses conversation content solely to provide the prompt insertion and export features described above. It does not transfer this content to third parties, does not use it for advertising, and does not use it for any purpose unrelated to the extension's single, user facing function.

Children

Sidecar is a professional tool intended for people using the Shopify admin. It is not directed at children and does not knowingly collect information from anyone.

Changes to this policy

If we change how Sidecar handles information, we will update this policy and revise the "Last updated" date above. Material changes will be reflected here before the corresponding version of the extension is published.
